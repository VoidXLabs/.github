### 🌟 核心项目：AI Mock Interviewer (AI 模拟面试官)

这是一个基于大语言模型（LLM）的多模态 AI 模拟面试平台。它能够模拟真实的面试场景，并为求职者提供全方位的复盘反馈。

#### ✨ 核心亮点
*   **🎭 多角色模拟**：支持大厂 HR、技术专家、架构师等多种面试官人格切换。
*   **🗣️ 实时音视频**：支持低延迟的语音对练与面部表情/眼神情绪分析。
*   **📊 维度深度评估**：面试结束后自动生成多维度诊断报告（包含代码能力、沟通表达、逻辑条理）。
*   **🎯 动态追问机制**：基于候选人的回答内容，AI 会进行硬核的“简历深挖”与“技术追问”。

#### 🛠️ 技术栈
`Next.js` `FastAPI` `Python` `LangChain` `WebRTC` `PostgreSQL` `OpenAI / DeepSeek API`
